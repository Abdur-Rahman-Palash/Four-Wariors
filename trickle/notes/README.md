# Creative Agency Portfolio Website

A modern, multi-page portfolio website showcasing digital marketing, graphic design, and web development services.

## Website Structure

### Pages
- **Home (index.html)** - Hero carousel, service highlights, stats section
- **About (about.html)** - Team introduction, company values
- **Services (services.html)** - Detailed service offerings
- **Portfolio (portfolio.html)** - Project showcase with filtering
- **Testimonials (testimonials.html)** - Client reviews and ratings
- **Contact (contact.html)** - Contact form and social links

## Key Features

- Responsive design optimized for all devices
- Hero carousel with auto-play (5-second intervals)
- Animated statistics section
- Project filtering by category (web, design, marketing)
- Floating WhatsApp and Telegram contact buttons
- Smooth hover effects and transitions
- SEO-optimized meta tags

## Design System

- **Primary Color:** `#2563eb` (Blue)
- **Secondary Color:** `#f8fafc` (Light Gray)
- **Typography:** Sans-serif with bold headings
- **Layout:** Grid-based with generous spacing

## Components

- Header - Responsive navigation with mobile menu
- Footer - Links and social media icons
- HeroCarousel - Auto-rotating slides
- ServiceCard - Service highlights
- StatsSection - Animated achievements
- TeamMember - Team profile cards
- ProjectCard - Portfolio items with hover effects
- TestimonialCard - Client reviews with ratings
- ContactForm - Message submission form
- FloatingContact - Fixed WhatsApp/Telegram buttons
