When updating the project

- Always check if the README.md in trickle/notes needs to be updated
- Update README.md when:
  - New pages or features are added
  - Design system changes (colors, fonts, layout)
  - New components are created
  - Project structure is modified
  - Contact information is updated
- Keep README.md concise and up-to-date
- Include the current date in "Last Updated" section
